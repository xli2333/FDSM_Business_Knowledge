# 本地启动与模拟用户体验说明

本文档用于本地启动前后端，并体验游客、免费会员、付费会员、管理员四类用户视角。

---

## 一、启动前准备

请确认：

- 已安装 Python 与 Node.js
- 已安装前端依赖：`frontend/node_modules` 已就绪
- 当前项目根目录为：`C:\Users\LXG\fdsmarticles`

---

## 二、启动后端

打开第一个 PowerShell 终端，执行：

```powershell
cd C:\Users\LXG\fdsmarticles
$env:DEV_AUTH_ENABLED='1'
$env:ADMIN_EMAILS='admin@example.com'
python -m uvicorn backend.main:app --host 127.0.0.1 --port 8000
```

说明：

- `DEV_AUTH_ENABLED=1` 用于开启本地模拟用户调试能力
- `ADMIN_EMAILS=admin@example.com` 用于让管理员模拟账号拥有后台权限
- 后端启动后默认地址为：`http://127.0.0.1:8000`

健康检查可访问：

```text
http://127.0.0.1:8000/
```

---

## 三、启动前端

打开第二个 PowerShell 终端，执行：

```powershell
cd C:\Users\LXG\fdsmarticles\frontend
$env:VITE_API_BASE_URL='http://127.0.0.1:8000/api'
npm run dev -- --host 127.0.0.1 --port 4173
```

说明：

- `VITE_API_BASE_URL` 明确指定前端请求本地后端
- 前端启动后访问地址为：`http://127.0.0.1:4173`

---

## 四、进入系统

浏览器打开：

```text
http://127.0.0.1:4173
```

登录页地址：

```text
http://127.0.0.1:4173/login
```

## 六、建议体验路径

### 游客视角

1. 打开首页
2. 浏览文章、专题、栏目
3. 打开付费文章，确认只能看到预览
4. 打开商业化页面与时间线、专题页

### 免费会员视角

1. 进入登录页
2. 选择 `reader@example.com`
3. 查看 `/me`
4. 测试收藏、点赞、关注、历史

### 付费会员视角

1. 进入登录页
2. 选择 `paid@example.com`
3. 打开付费文章
4. 确认可以查看完整正文、音视频等付费内容

### 管理员视角

1. 进入登录页
2. 选择 `admin@example.com`
3. 依次访问：
   - `/admin`
   - `/admin/memberships`
   - `/editorial`
   - `/commercial/leads`
   - `/media-studio`

## 七、停止服务

前端和后端都可在各自终端中按：

```text
Ctrl + C
```

停止运行。

---

## 八、补充说明

- 当前环境下如遇到英文翻译接口不可用，系统会返回明确的服务不可用提示，不会导致整站崩溃。
- 如果前端打不开数据，先确认后端是否已启动在 `8000` 端口。
- 如果登录页没有出现模拟账号，先确认后端启动时是否设置了：

```powershell
$env:DEV_AUTH_ENABLED='1'
```
