# 当前项目资料目录说明

本目录用于集中管理当前知识库项目的“非运行代码资料”，避免品牌素材、交付文档和运行链路混在根目录。

## 目录结构

- `docs/`
  当前项目的说明文档、运行文档和技术实现文档。
- `branding/`
  品牌图形、Logo 备份和视觉素材。
- `delivery/`
  对外交付说明、作品说明等交付文档。
