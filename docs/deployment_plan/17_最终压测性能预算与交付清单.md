# 17 最终压测、性能预算与交付清单

## 推荐私有云配置

第一阶段单机私有云建议：

- CPU：4 vCPU 起步，推荐 8 vCPU。
- 内存：16 GB 起步，推荐 32 GB。FAISS/RAG、本地 SQLite、Gunicorn worker、Worker 和前端 Nginx 都在同机时，16 GB 是下限。
- 磁盘：系统盘 80 GB；数据盘 500 GB SSD 起步，建议单独挂载到 `/srv/fdsm/app/data` 与 `/srv/fdsm/app/backups`。
- 带宽：10 Mbps 起步；如果音视频访问变多，建议 20-50 Mbps 或接 OSS/CDN。
- 备份：本机每日 SQLite 备份，至少每周异地同步一次。
- 首页公开 feed：默认开启 `HOME_FEED_CACHE_TTL_SECONDS=30` 的短缓存，用来承接高频读流量；内容运营更新允许最多 30 秒自然刷新，正式上线可按运营实时性要求调到 10-60 秒。

升级触发点：

- SQLite 数据库超过 5 GB 或持续写 QPS 超过 50，迁移 PostgreSQL。
- 单机 CPU 长期超过 70%，增加机器或拆 Worker。
- 内存长期超过 80%，拆 RAG/FAISS 或提高规格。
- 音视频流量吃满带宽，迁移 OSS/CDN。
