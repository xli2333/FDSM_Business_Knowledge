# 服务器与 Gemini AI 接口申请附件

## 一、系统说明

本项目建设的是“复旦管院智识库系统”，用于学院知识内容的统一管理、检索、发布和智能问答。

系统主要沉淀学院文章、研究内容、专题知识库、音视频材料等资料，支持用户端浏览检索、编辑后台内容管理、AI 摘要、英文翻译、知识库问答和内容推荐等功能。

系统正式运行需要申请生产服务器资源和 Gemini AI 接口。

## 二、服务器申请清单

| 项目 | 建议配置 | 说明 |
|---|---|---|
| 服务器类型 | 私有云服务器或学校统一云服务器 | 用于部署正式系统 |
| 操作系统 | Ubuntu Server 22.04 LTS 或 Ubuntu Server 24.04 LTS | 建议使用 Linux 服务器，不建议使用 Windows Server |
| CPU | 8 vCPU 推荐，4 vCPU 起步 | 支持前后端服务、检索、后台任务和内容处理 |
| 内存 | 32 GB 推荐，16 GB 起步 | 支持知识库检索、缓存、异步任务和系统稳定运行 |
| 公网访问 | 需要 | 用于正式网站访问 |
| 域名与 HTTPS | 需要 | 用于绑定正式访问域名和配置 HTTPS 证书 |
| 备份能力 | 需要 | 用于系统数据和内容资产的定期备份 |

## 三、Gemini AI 接口申请清单

| 接口类型 | 建议接口 | 用途 |
|---|---|---|
| Gemini 文本生成接口 | Gemini Flash 系列模型 | AI 问答、文章摘要、内容改写、英文翻译、音视频总结 |
| Gemini Embedding 接口 | Gemini Embedding 系列模型 | 知识库语义检索、RAG 问答、相似内容召回 |
| 生产环境 API Key | Gemini API Key | 系统正式运行调用，不在前端暴露，仅保存在服务器环境变量中 |
| 调用额度 | 建议支持稳定生产调用 | 用于后台批量处理、知识库更新和用户问答 |

## 四、申请说明

服务器用于运行复旦管院智识库系统的前端网站、后端服务、数据库、知识库索引、异步任务和备份服务。

Gemini AI 接口用于系统中的智能问答、内容摘要、英文翻译、标签生成、音视频内容总结和知识库语义检索等功能。

以上资源为系统上线和持续运行所需的基础条件。
