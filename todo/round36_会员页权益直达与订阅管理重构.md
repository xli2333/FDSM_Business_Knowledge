# Round36 会员页权益直达与订阅管理重构

## 目标

让会员页从“方案售卖页”调整为“身份驱动页”：
- 已经是会员时，不再优先展示大块方案卡片
- 免费会员、付费会员、管理员进入会员页后，优先看到已解锁权益与可去的内容入口
- 将订阅、账单、续费/升级动作收拢到“管理订阅”区域
- 保持游客仍可查看方案与支付环境

- [x] P0 | 会员页 | 复核现有会员页信息层级与已订阅用户误导点
- [x] P0 | 会员页 | 建立游客与已订阅用户的分流展示逻辑
- [x] P0 | 会员权益 | 为免费会员、付费会员、管理员输出直达权益面板
- [x] P0 | 订阅管理 | 将当前方案、状态、账单记录、升级/续费动作收拢为管理订阅区
- [x] P1 | 订阅管理 | 弱化已订阅用户的售卖型方案卡，避免与当前身份冲突
- [x] P1 | 文案 | 收口会员页中文/英文文案，统一为用户口径

- [x] P1 | 回填 | 按完成情况逐项勾选并记录本轮结论
