## P0 正文区验收标准
- [ ] 标题下方元信息行保持站内阅读页结构，不得变成公众号整页壳
- [ ] 正文区必须仍在左栏，不得回退成单栏阅读页
- [ ] 正文区必须使用真实复旦模板 HTML，而不是普通 Markdown 文本块
- [ ] 正文首屏要能看到“正文”区块起始，而不是被超长摘要完全挤掉
- [ ] 正文中的强调句、章节标题、节奏留白要明显接近复旦公众号自动排版
- [ ] 正文不得引入广告、订阅引导、扫码引导、转载留言等尾部垃圾
- [ ] 正文不得改写原文意思，不得重写正文论点

## P0 摘要区验收标准
- [ ] 摘要区必须使用与正文同一视觉体系的复旦模板风格
- [ ] 摘要区不能出现明显的“模型原始列表稿”观感
- [ ] 摘要区可折叠，但首屏必须保持可读并让出正文起始区空间
- [ ] 摘要区中英文都要执行同一套显示清洁和版式规范
- [ ] 摘要区不得带尾部运营垃圾信息
