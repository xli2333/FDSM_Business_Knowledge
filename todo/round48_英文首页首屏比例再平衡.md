# Round48 英文首页首屏比例再平衡
## 目标

解决英文首页首屏左栏过宽、右栏过窄的问题，只调整英文首页分支，不影响中文页。

## 原子任务

- [x] P0 | 英文首屏比例 | 调整英文首页首屏左右列比例，让右侧 Editor's Picks 更宽
- [x] P0 | 英文 Hero | 同步收紧英文标题字号与字距，避免左栏过满且保持第二行单行
- [x] P1 | 验收 | 运行前端构建与 Lint，确认英文首页改动通过
- [x] P1 | 回填 | 记录本轮结论并结束该轮

## 本轮结论

- 英文首页首屏已从明显左重右轻改为更均衡的双列比例
- 右侧 Editor's Picks 在常见桌面宽度下会更宽，阅读压力更小
- 英文 Hero 字号同步下调，避免左栏视觉过满，同时继续保持第二行单行
