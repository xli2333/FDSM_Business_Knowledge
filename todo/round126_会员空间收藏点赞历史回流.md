# Round 126：会员空间收藏点赞历史回流

## 对齐结论
- [x] 会员空间里的“收藏 / 点赞 / 历史”不能只停留在数字和说明层，必须成为可点击的回流入口。
- [x] 用户从会员空间点击某一类资产后，应直接进入资料库页对应的文章流，而不是先到一个泛入口再自己找。
- [x] 资料库页需要支持按 `bookmarks / likes / history` 定位和切换，明确展示对应文章流。
- [x] 文章流里的卡片继续保持可点击，用户可以从资料库回到文章详情页。
- [x] 本轮不重做用户资产数据模型，优先收口前端入口、筛选交互与真实验收。
- [x] 全部 todo 与代码改动保持 UTF-8 中文，不引入中文编码错误或 JSX 错误。
