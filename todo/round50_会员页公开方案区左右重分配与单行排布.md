# Round50 会员页公开方案区左右重分配与单行排布
## 目标

解决会员页公开方案区右侧过挤的问题，把左列收窄、右列加宽，并让桌面端方案卡片单行排开。

- [x] P0 | 卡片收口 | 适当压缩卡片内边距、价格字号和说明行高，缓解右侧拥挤

- [x] P1 | 回填 | 记录本轮结论

- 会员页公开方案区已调整为左窄右宽
