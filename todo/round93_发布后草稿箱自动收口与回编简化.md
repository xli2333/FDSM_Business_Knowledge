# Round 93：发布后草稿箱自动收口与回编简化
## 对齐规则
- [x] 规则 1：未发布稿保留“删除草稿”，可以直接删。
- [x] 规则 2：一旦正式发布成功，该稿件要自动从草稿箱消失，不再出现“移出草稿箱 / 恢复到草稿箱”这类手工操作。
- [x] 规则 3：已发布文章如果要继续修改，只保留“正文页一键打回草稿箱修改”这一条入口。
- [x] 对用户语义来说是“已发布自动从草稿箱删除”；内部若为了回编链路保留最小关联记录，可以继续保留，但不能再暴露成手工移出/移入操作。
- [x] 全部 todo 文案保持 UTF-8 中文，避免中文乱码。
